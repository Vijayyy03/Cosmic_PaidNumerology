# Cosmic Paid Numerology

A modern, Ancient Vedic-themed numerology website built with React, Vite, and Tailwind CSS.

## Features

- **Ancient Vedic Theme:** Authentic design with warm gold, deep maroon, and cream colors.
- **Numerology Calculator:** Calculates Life Path, Destiny, Soul Urge, and Personality numbers.
- **Detailed Reports:** Provides in-depth interpretations based on Vedic numerology principles.
- **Conversion Optimization:** Strategic funnel from Landing Page -> Form -> Report.
- **Responsive Design:** Fully mobile-optimized interface.

## Project Structure

- `src/pages/`: Main page components (LandingPage, FormPage, ReportPage)
- `src/components/`: Reusable UI components
- `src/utils/`: Numerology calculation logic
- `src/data/`: Interpretation texts and constants
- `src/context/`: React Context for state management

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Vijayyy03/Cosmic_PaidNumerology.git
   ```

2. Navigate to the frontend directory:
   ```bash
   cd Cosmic_PaidNumerology/frontend
   ```
   *Note: If you have just downloaded the zip, navigate to the extracted folder.*

3. Install dependencies:
   ```bash
   npm install
   ```

### Running Development Server

To start the development server with hot reload:

```bash
npm run dev
```

The application will be available at `http://localhost:5174`.

### Building for Production

To create an optimized production build:

```bash
npm run build
```

The build artifacts will be stored in the `dist/` directory.

### Preview Production Build

To preview the production build locally:

```bash
npm run preview
```

## Technologies Used

- **React:** UI Library
- **Vite:** Build tool and dev server
- **Tailwind CSS:** Utility-first CSS framework
- **Framer Motion:** Animation library
- **React Router:** Client-side routing
